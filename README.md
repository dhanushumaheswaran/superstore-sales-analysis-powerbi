# Sales Analysis Dashboard (Power BI)

## Project Overview
This project presents an interactive Sales Analysis Dashboard built using Power BI. It provides insights into sales performance across different regions, categories, segments, and time periods to support data-driven decision-making.

---

## Tools & Technologies
- Power BI  
- DAX (Data Analysis Expressions)  
- Data Modeling  

---

## Dashboard Features

  ### Time Analysis
  - Sales trends by Day and Month
  - Year-wise filtering (2018–2021)

  ### Regional Analysis
  - Sales distribution across West, East, Central, and South regions

  ### Category & Sub-Category Insights
  - Sales by Category (Technology, Furniture, Office Supplies)
  - Sub-category performance comparison

  ### Customer Segmentation
  - Sales split by Consumer, Corporate, and Home Office segments

  ### KPI Metrics
  - Total Sales: 2.29M+
  - Total Profit: 286.40K
  - Total Loss: -156.13K
  - Total Quantity Sold: 38K

  ### Interactive Features
  - Year slicer (2018–2021)
  - Dynamic filtering across visuals
  - “Clear All” button for quick reset

---

## Key Insights

- West region contributes the highest share of total sales  
- Technology category generates the highest revenue  
- Discounts significantly impact profitability (visible loss segment)  
- Sales show variation across months with peak periods
